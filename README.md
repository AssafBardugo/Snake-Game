
# Snake Game in C using SDL

Screenshot from the game:
<img src="Screenshot.png" alt="Screenshot" width="425" height="450" style="float: right; margin-left: 100px;">

Compilation and running:
```
$ ./build.sh
$ ./snake
```

Control keys:
```
j - left
l - right
i - up
k - down

Space - pause
Esc - quit
```
